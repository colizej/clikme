### URL структура вендоров

OpenCart: `/vendor_slug/product_slug`  
Django: `/vendors/vendor_slug/`

**Редиректы для вендоров:**
```python
# Пример для vendor elki
'/elki-restaurant-russian-cuisine/каша-гречневая' → '/vendors/elki-restaurant-russian-cuisine/'
```

> ⚠️ В Django вендоры отображаются как одна страница с products. Если продукты были важны для SEO (имели отдельные слаги), нужно решить:
> 1. Объединить в страницу вендора (текущее решение)
> 2. Сделать отдельные страницы продуктов (потребует доработки)

## Структура URL

### OpenCart (старое)
```
/{article_slug}/                    # Статьи блога
/{vendor_slug}/{product_slug}/     # Продукты вендоров
/category/{category_slug}/          # Категории
/information/{slug}/                # Информационные страницы
```

### Django (новое)
```
/{slug}/                            # Статьи блога
/vendors/                           # Список вендоров
/vendors/{slug}/                    # Страница вендора + продукты
/news/                              # Новости (новый раздел)
/privacy/                           # Политика конфиденциальности
/contacts/                          # Контакты
```
